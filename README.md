# World-Population-Analysis
This project presents a comprehensive analysis of world population data using Python, JupyterLab, Pandas, and Matplotlib. It explores key population trends by identifying the top 10 most and least populated countries, comparing population growth rates, analyzing population distribution across continents, and creating clear visualization.
# World Population Analysis

This project analyzes world population data using Python and Jupyter Notebook.

## Features
- Top 10 Most Populated Countries
- Top 10 Least Populated Countries
- Population Growth Analysis
- Population by Continent
- Bar Charts and Pie Charts

## Tools Used
- Python
- Pandas
- Matplotlib
- JupyterLab
